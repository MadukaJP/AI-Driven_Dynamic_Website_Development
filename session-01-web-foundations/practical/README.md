# Practical: Your First Web Page

Build a simple "About Me" page from scratch.

## Steps

1. Create a new file `about-me.html` in this folder.
2. Start from the boilerplate in `../snippets/boilerplate.html`.
3. Set the `<title>` to your name.
4. Add:
   - An `<h1>` with your name
   - A `<p>` with 2-3 sentences about yourself
   - A second `<p>` listing 3 things you want to learn in this course
5. Open it with Live Server and check it renders correctly.
6. Open the browser DevTools (`F12` or right-click → Inspect) and look at the **Elements** tab — confirm your HTML shows up exactly as you wrote it.
