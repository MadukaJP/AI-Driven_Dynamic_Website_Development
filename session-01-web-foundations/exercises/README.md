# Session 1 Exercises

Attempt these before checking `../solutions/`.

## Exercise 1: Fix the broken page

The file `broken-page.html` has 3 mistakes preventing it from working correctly.
Find and fix them.

## Exercise 2: Explain in your own words

In a file `notes.md`, answer in 1-2 sentences each:
1. What is the job of HTML vs CSS vs JavaScript?
2. Why don't we use XHTML anymore?
3. What does caniuse.com help you decide?

## Exercise 3: AI cross-check

Ask your AI assistant: "What's the difference between HTML5 and HTML4?"
Paste its answer into `notes.md`, and add one sentence noting anything
it got wrong, oversimplified, or that you'd want to verify.
