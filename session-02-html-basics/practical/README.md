# Practical: A Structured Recipe Page

Build `recipe.html` with proper HTML structure and accessibility in mind.

## Steps

1. Start from Session 1's boilerplate.
2. Add:
   - One `<h1>` — the recipe name
   - An `<img>` with a meaningful `alt` describing the dish
   - An `<h2>` "Ingredients" followed by a `<ul>` list
   - An `<h2>` "Steps" followed by an `<ol>` list
   - A `<div class="card">` wrapping a short note about prep time
3. Open DevTools → Elements tab and confirm your tag nesting is correct
   (no unclosed tags, proper indentation).
4. Double check: is there only one `<h1>`? Are your headings in order
   with no skipped levels?
