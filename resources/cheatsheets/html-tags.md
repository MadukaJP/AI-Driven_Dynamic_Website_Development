# HTML Tags Cheatsheet

## Structure
| Tag | Use |
|---|---|
| `<header>` | Page/section intro |
| `<nav>` | Navigation links |
| `<main>` | Primary content (one per page) |
| `<article>` | Self-contained content |
| `<section>` | Thematic grouping |
| `<aside>` | Tangential content |
| `<footer>` | Closing content |
| `<div>` | Generic block, no semantic meaning |
| `<span>` | Generic inline, no semantic meaning |

## Text
| Tag | Use |
|---|---|
| `<h1>`–`<h6>` | Headings, in order |
| `<p>` | Paragraph |
| `<strong>` | Important text (bold) |
| `<em>` | Emphasized text (italic) |
| `<mark>` | Highlighted text |
| `<blockquote>` | Block quotation |
| `<code>` / `<pre>` | Code / preformatted text |

## Lists
| Tag | Use |
|---|---|
| `<ul>` / `<li>` | Unordered list |
| `<ol>` / `<li>` | Ordered list |
| `<dl>` / `<dt>` / `<dd>` | Description list |

## Media & links
| Tag | Use |
|---|---|
| `<a href="">` | Link |
| `<img src="" alt="">` | Image (alt required) |
| `<video controls>` | Video |
| `<audio controls>` | Audio |
| `<picture>` | Multiple image sources |

## Forms
| Tag | Use |
|---|---|
| `<form>` | Form container |
| `<label for="">` | Label paired to an input |
| `<input type="">` | Input field (see input types cheatsheet) |
| `<textarea>` | Multi-line text |
| `<select>` / `<option>` | Dropdown |
| `<button>` | Button |
