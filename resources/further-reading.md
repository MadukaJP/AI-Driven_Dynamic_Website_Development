# Further Reading

## Reference documentation (bookmark these)

- [MDN Web Docs](https://developer.mozilla.org/) — the single best reference for HTML, CSS, and JavaScript. When in doubt, check here.
- [caniuse.com](https://caniuse.com) — browser support tables for any feature
- [web.dev](https://web.dev) — Google's guidance on performance, PWAs, and modern best practices

## Books (from the original course's library references, still solid for HTML/CSS fundamentals)

- *HTML5: Up and Running* by Mark Pilgrim
- *Introducing HTML5 (Voices That Matter)* by Bruce Lawson
- *Head First HTML5 Programming: Building Web Apps with JavaScript* by Eric Freeman, Elisabeth Robson

## Modern additions worth exploring after this course

- *Eloquent JavaScript* by Marijn Haverbeke (free online: https://eloquentjavascript.net) — a genuinely excellent, modern JS deep-dive
- [JavaScript.info](https://javascript.info) — a thorough, modern, free JS reference and tutorial
- [Tailwind CSS docs](https://tailwindcss.com/docs) — if you want to explore utility-first CSS further
- [Three.js docs](https://threejs.org/docs) — the natural next step after Canvas (Session 17) if you're interested in 3D/graphics
- [Anthropic API docs](https://docs.claude.com) and [OpenAI API docs](https://platform.openai.com/docs) — for going deeper on Session 18's AI integration
